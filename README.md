# Banking Management System 💳

A secure and modern Banking Management System developed using Spring Boot.  
This project helps manage banking operations such as account handling, transactions, and user authentication.

---

## 🚀 Features

- User Authentication
- Account Management
- Deposit & Withdraw
- Money Transfers
- Transaction History
- Secure REST APIs

---

## 🛠 Technologies Used

- Java
- Spring Boot
- Spring Security
- MySQL
- Hibernate / JPA
- Maven
